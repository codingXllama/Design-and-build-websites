Purpose: To Display the List Types in HTML

These lists include Ordered,Unordered,and Definition Lists

Note All list types begin with the li tag


Ordered List:When Each item of the list is Numbered

Element Used: Hover Me

Use Cases: steps for a receipt, a to-do list, class notes,etc.

Example: Suppose you have a shopping list as follows:

  1. Get 12 Eggs
  2. Buy Milk
  3. Buy Cheese
  4. Buy Soda

Unordered lists: A List that contains list items that begins with a bullet point rather than a number.

Element Used: Hover Me

Use Cases: steps for a receipt, a to-do list, class notes,etc.

Example: Suppose you are taking notes for a meeting:

  • Add CSS to my HTML
  • Comment my code
  • Check my peers code

Definition List:When Each item(term) is given a specific definition

Element Used: Hover Me

Use Cases: Making a new dictionary of words to understand.

Example: Suppose you're asked to find the definition of life:

Life:
A big ball of problems, ready to be solved.
